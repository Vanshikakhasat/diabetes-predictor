# 🩺 AI Diabetes Prediction System
A Machine Learning web application built using Streamlit that predicts diabetes using Decision Tree and K-Nearest Neighbors (KNN) algorithms.
## 🚀 Features

- 📂 Upload dataset (CSV or Excel)
- 🧠 Automatic target column detection
- ⚙️ Data preprocessing (missing values handling, scaling)
- 🤖 Machine Learning models:
  - Decision Tree
  - K-Nearest Neighbors (KNN)
- 📊 Model comparison using:
  - Accuracy
  - ROC Curve
  - Confusion Matrix
- 📈 Interactive visualizations
- 🖥️ User-friendly interface with Streamlit
  ---

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas & NumPy
- Matplotlib & Seaborn

---

### ▶️ How to Run Locally
## 1. Clone the repository:
```   

git clone https://github.com/your-username/diabetes-predictor.git

cd diabetes-predictor
```

## 2. Install dependencies:
```
pip install -r requirements.txt
```
## 3. Run the app:
```
python -m streamlit run app.py
```

📊 Dataset

- You can upload your own diabetes dataset (CSV or Excel)

- The system automatically detects suitable target columns

🧠 Project Description

This project is designed to analyze diabetes datasets and compare machine learning models to predict whether a patient is diabetic or not. It provides an interactive interface for users to upload datasets, train models, and visualize results.
